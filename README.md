# Sistema de Notas de Alunos

Sistema desenvolvido para gerenciar notas de alunos e gerar relatórios de desempenho, classificando-os como **aprovados** ou **reprovados**.

---

## Funcionalidades

- Cadastro de alunos  
- Registro de notas  
- Cálculo automático da média  
- Classificação de alunos:
  - Aprovados  
  - Reprovados  
- Geração de relatório final  

---

## Regras_de_Negócio

- A média do aluno é calculada com base nas notas inseridas  
- Critério de aprovação:
  - Média **maior ou igual a 6** → **Aprovado**
  - Média **menor que 6** → **Reprovado**

---

## Como funciona

1. O usuário cadastra o aluno  
2. Insere as notas  
3. O sistema calcula a média automaticamente  
4. O aluno é classificado como aprovado ou reprovado  
5. Ao final, é gerado um relatório com todos os alunos  

---

## Exemplo de saída

    Aluno: João
    Média: 7.5
    Status: Aprovado

    Aluno: Maria
    Média: 5.0
    Status: Reprovado
